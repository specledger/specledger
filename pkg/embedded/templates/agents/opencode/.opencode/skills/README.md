# OpenCode Skills

Place your OpenCode skill definitions here.

Skills are reusable capabilities that can be invoked by OpenCode.

## Structure

```
skills/
├── my-skill/
│   └── skill.md
└── another-skill/
    └── skill.md
```

## Example Skill

```markdown
# My Skill

Description of the skill.

## When to Use

Conditions for when this skill should be used.

## Instructions

Steps to execute the skill.
```
