## Technical Decisions

### 1. Cross-Platform Path Handling
- **Decision**: Use filepath.Join() for all path operations
- **Rationale**: Ensures correct separators on macOS, Linux, Windows
- **Implementation**: All paths.go functions use filepath.Join()

### 2. Error Handling
- **Decision**: Return structured errors with context
- **Rationale**: Better debugging and user experience
- **Implementation**: fmt.Errorf with %w for error wrapping

### 3. JSON Output Format
- **Decision**: Uppercase field names (FEATURE_DIR vs feature_dir)
- **Rationale**: Match existing bash script output format for compatibility
- **Implementation**: JSON struct tags with uppercase names

### 4. Stop-Word Filtering
- **Decision**: Use map for O(1) lookup
- **Rationale**: Efficient filtering of common words
- **Implementation**: StopWords map[string]bool in branch.go

### 5. Acronym Preservation
- **Decision**: Detect uppercase patterns and preserve
- **Rationale**: Keep OAuth2, API, JWT readable
- **Implementation**: Regex pattern matching in PreserveAcronyms()
