## Technical Decisions

### 1. filepath.Join() Everywhere
- **Decision**: Use filepath.Join() for ALL path operations
- **Rationale**: Automatic separator handling, cross-platform
- **Implementation**: 22 calls verified across all files

### 2. No Shell Invocation
- **Decision**: Never invoke shell commands
- **Rationale**: Windows doesn't have bash by default
- **Implementation**: Pure Go, no exec.Command for git/text ops

### 3. JSON Encoding Consistency
- **Decision**: Same encoder settings across all commands
- **Rationale**: Consistent output format for AI agents
- **Implementation**: SetEscapeHTML(false), SetIndent("", "  ")

### 4. Manual Windows Testing
- **Decision**: Create comprehensive guide instead of CI
- **Rationale**: No Windows CI available, guide enables user testing
- **Implementation**: windows-testing-guide.md with 10+ test scenarios
